Question-1: What are the different ways to select an element in the DOM?

Elements can be selected using different methods in JavaScript, such as:
- getElementById("id") to select an element by its unique ID.
- getElementsByClassName("class") to select all elements with a class.
- getElementsByTagName("tag") to get elements by tag name.
- querySelector(".class or #id or tag") for the first matching element.
- querySelectorAll(".class or tag") to select all matching elements.
- getElementsByName("name") to select elements by name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent?

- innerHTML: Retrieves or sets the entire HTML content of an element.
- innerText: Retrieves or sets only the visible text inside an element.
- textContent: Retrieves or sets the text inside an element, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is placed on a parent element to handle events on its child elements dynamically. This reduces the number of event listeners and improves performance.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates upward through its parent elements. This allows parent elements to listen for events on their children without attaching individual event listeners.

Question-5: How do you create, add, and remove elements using JavaScript?

- To create an element: document.createElement("tag").
- To add an element: parentElement.appendChild(childElement) or insertBefore(newElement, referenceElement).
- To remove an element: parentElement.removeChild(childElement).
